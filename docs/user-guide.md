## Getting Started

### First Time Access
1. Navigate to the application URL
2. Click "Sign Up" if you're a new user
3. Enter your email address
4. Complete registration with your full name and password
5. Verify your email address if required

### Logging In
1. Visit the login page
2. Enter your registered email
3. Provide your password
4. Click "Sign In"

### Accepting Invitations
1. Click the invitation link in your email
2. If not registered:
   - Complete the registration process
   - You'll be automatically added to the project
3. If already registered:
   - Log in with your credentials
   - You'll be redirected to the project

## Projects

### Dashboard Navigation
- View all your projects
- See recent activity
- Access project invitations
- Quick-access to favorite projects

### Creating a Project
1. Click "New Project" on the dashboard
2. Enter project details:
   - Project name
   - Description
   - Key/Identifier
3. Configure initial settings
4. Invite team members (optional)

### Project Views
- **Backlog**: All project tasks
- **Sprints**: Active and planned sprints
- **Board**: Kanban-style task management
- **Settings**: Project configuration

## Task Management

### Creating Tasks
1. Navigate to project backlog
2. Click "Create Task"
3. Fill in task details:
   - Title
   - Description
   - Priority
   - Due date (optional)
   - Assignee (optional)
4. Save the task

### Managing Tasks
- **Status Updates**:
  1. Drag and drop between columns
  2. Or use status dropdown
- **Assignment**:
  1. Click assign button
  2. Select team member
- **Priority**:
  - Set task priority level
  - Reorder tasks within columns

### Task Details
- View full details by clicking task
- Add comments
- Attach files (if supported)
- Track time spent
- View task history

## Team Collaboration

### Team Management
- **Inviting Members**:
  1. Go to Project Settings > Team
  2. Click "Invite Member"
  3. Enter email address
  4. Select role
  5. Send invitation

### Roles and Permissions
- **Owner**:
  - Full project control
  - Manage all settings
  - Delete project
- **Administrator**:
  - Manage team members
  - Configure project settings
  - Full task management
- **Manager**:
  - Create and manage tasks
  - Assign team members
  - View all project data
- **Member**:
  - View assigned tasks
  - Update task status
  - Add comments

## Troubleshooting

### Common Issues
1. **Can't Access Project**
   - Check your permissions
   - Verify project URL
   - Contact project administrator

2. **Email Notifications**
   - Check spam folder
   - Verify email settings
   - Update notification preferences

3. **Task Updates Not Showing**
   - Refresh the page
   - Clear browser cache
   - Check internet connection
